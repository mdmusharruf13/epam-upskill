problem statement :

Write a program to check whether number is positive or negative.

Code : 

            function checkPositiveNegative(num){
                if(num > 0){
                    console.log(`It is a Positive Number`);
                }
                else if(num < 0){
                    console.log(`It is a Negative Number`);
                }
            }
            checkPositiveNegative(598);

Output :

Check in the console.

Note : you can also call this function by passing any number in the console.
checkPositiveNegative(any number);
